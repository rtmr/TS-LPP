# TS-LPP
Two Step Locality Preserving Projections (TS-LPP)

# Requirements
Python >= 3.7

numpy >= 1.16.2

scikit-learn >= 0.20.3

scipy >= 1.4.1

matplotlib >= 2.2.3

# Installation
Download or clone the github repository, e.g. git clone https://github.com/rtmr/TS-LPP

# Usage

## **Parameters:**

n_components

The reduced dimension in the final step is defined by n_components.
